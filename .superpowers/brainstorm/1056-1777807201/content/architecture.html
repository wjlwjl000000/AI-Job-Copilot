系统架构总览

FastAPI + Vue.js — 模块划分与数据流

五大功能模块

1. 简历优化

上传 → 解析 → 多版本管理 → AI 优化建议 → 岗位匹配度评分

2. 职位匹配

简历向量化 → 搜索/爬取职位 → 语义匹配 → 智能推荐排序

3. 投递追踪

投递记录 → 状态看板 → AI 总结 → 时间线可视化

4. 面试备战

JD 解析 → 生成问题 → 模拟面试 → 回答评估 → 改进建议

5. 情感支持 (贯穿全流程)

RAG 检索相似经历 → Agent 生成个性化鼓励 → 关键节点推送（匹配度低/被拒/面试失败/拿到offer）

技术分层

前端层 — Vue.js 3 + Vite + Element Plus + Pinia
API 网关
Nginx / Caddy
后端层 — FastAPI + LangChain + Celery (异步任务)
PostgreSQL
业务数据
pgvector
向量存储/RAG
Redis
缓存/任务队列
AI 层 — DeepSeek API / 通义千问 API (可切换)

核心数据流

📄 用户上传简历 → 🔍 简历解析提取技能/经历 → 🧠 向量化存入 pgvector → 🎯 匹配职位 (语义相似度) → 📊 展示匹配度 + 优化建议 → 📬 投递后跟踪状态 → 🎤 面试准备 + AI 模拟 → 💬 全流程 RAG 情感支持
