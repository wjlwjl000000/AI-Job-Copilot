# Answer Evaluation Rubric

## Scoring Dimensions

### Accuracy (准确性) — Weight 0.40

| Score | Description |
|-------|-------------|
| 9-10 | Technically flawless, all key points correct |
| 7-8 | Mostly correct, 1-2 minor errors |
| 5-6 | Partially correct, some misunderstandings |
| 3-4 | Several major errors |
| 0-2 | Fundamentally wrong |

### Completeness (完整性) — Weight 0.35

| Score | Description |
|-------|-------------|
| 9-10 | Covers all expected points + bonus insights |
| 7-8 | Covers most expected points |
| 5-6 | Covers ~half the expected points |
| 3-4 | Missing most key points |
| 0-2 | Barely addresses the question |

### Clarity (表达清晰度) — Weight 0.25

| Score | Description |
|-------|-------------|
| 9-10 | Structured (STAR/other), concise, logical flow |
| 7-8 | Mostly clear, slightly rambling |
| 5-6 | Understandable but disorganized |
| 3-4 | Confusing, hard to follow |
| 0-2 | Incoherent |

## Overall Score Formula

`overall = 0.40 * accuracy + 0.35 * completeness + 0.25 * clarity`

## Feedback Structure

1. **Acknowledge strengths** (1 sentence) — what was good
2. **Point out gaps** (1-2 sentences) — what was missing or wrong
3. **Suggest improvement** (1-2 sentences) — how to answer better
4. **Provide example** — an improved version of the answer

## Tips
- Keep feedback constructive, not critical
- Score below 5 → consider triggering support agent for encouragement
- Focus on 1-2 actionable suggestions, not a laundry list
