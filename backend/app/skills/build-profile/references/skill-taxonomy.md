# Skill Taxonomy & Scoring Metrics

## Skill Level Criteria

| Level | Criteria | Keywords |
|-------|----------|----------|
| **初级** | Can use with guidance. < 1 year hands-on. | "了解" "接触过" "有基础" |
| **中级** | Independent daily use. 1-3 years experience. | "熟练使用" "日常开发" |
| **高级** | Deep understanding, can teach others. 3-5 years. | "精通" "主导过" "架构设计" |
| **专家** | Thought leadership, open source contributions. 5+ years. | "深入研究" "开源贡献" "技术分享" |

## Competitiveness Score

Scale 0.0-1.0 based on:
- Skill depth (avg level across top 5 skills)
- Experience relevance (years in target industry)
- Project complexity (team size, impact scope)

## Market Match Score

Scale 0.0-1.0 based on:
- Skill demand alignment (how in-demand are user's skills)
- Geographic match (target city vs job market)
- Salary alignment (expectation vs market range)

## Completeness Score

Scale 0.0-1.0 based on:
- Has education field
- Has work experience entries
- Has 5+ skill tags
- Has project examples
- Has target preferences
